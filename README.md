# Best Cars Dealership - Full Stack Capstone Project

## Repository Name
xrwvm-fullstack_developer_capstone

## Project Name
Best Cars Dealership Web Application

## Description
A full-stack web application built with Django, React, Node.js, MongoDB, and Flask.
Users can browse dealerships, view and post reviews, and administrators can manage car inventory.

## Technologies Used
- Frontend: React.js
- Backend: Django, Node.js, Express
- Database: SQLite, MongoDB
- Sentiment Analysis: Flask + NLTK
- Deployment: Docker, Railway
- CI/CD: GitHub Actions
